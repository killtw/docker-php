# PHP Docker Image

Alpine based PHP Docker image with extensions and composer.

## Extensions

* MySQL
* Opcache
* Redis
* GD
* Sockets

## Composer packages
* [hirak/prestissimo](https://github.com/hirak/prestissimo)
